# CSP3341 Java Preliminary Project

This repository contains a preliminary Java application developed for CSP3341.

It demonstrates:
- Java environment setup on macOS
- Compilation and execution using the JVM
- Basic Java program structure using a console application (HelloWorld)

**Language:** Java (OpenJDK 25 LTS Temurin)  
**IDE:** Visual Studio Code  
**DevOps:** Git and GitHub
